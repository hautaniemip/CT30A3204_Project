# CT30A3204_Project

## Technologies used

* MongoDB
* TypeScript
* NodeJS
* React

## Instalation

1. Run `npm install`
2. Run `npm run build`

## Running

This requires a running MongoDB instance

1. Define enviroment variables in  `server/.env` (see `server/.env.example`)
2. Define `NODE_ENV='production'` in `server/.env` before running production build
3. Run `npm start`
